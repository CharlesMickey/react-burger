### Запуск приложения
Скачать или склонировать репозиторий
```sh
# установка зависимостей
$ npm install

# запуск сервера на localhost:3000
$ npm run start

# сборка в продакшн
$ npm run build

# запуск тестирования приложения (Cypress)
$ npm run cypress:open

# запуск тестирования приложения
$ npm run test
```
